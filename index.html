1. Mengapa menggunakan HTML, CSS serta JavaScript dalam pemrograman web?

  • HTML (HyperText Markup Language)
    Fungsi utama HTML adalah untuk membuat struktur dasar sebuah halaman web. HTML digunakan untuk mendefinisikan elemen-elemen pada halaman web seperti teks, gambar, tabel, form, dan lainnya. HTML adalah pondasi dari sebuah halaman web dan wajib digunakan dalam pengembangan web.
  • CSS (Cascading Style Sheets)
    CSS digunakan untuk mengatur tampilan dan tata letak (layout) dari elemen-elemen HTML. Dengan CSS, developer dapat mengatur warna, font, margin, padding, serta tata letak responsif untuk berbagai perangkat. CSS memisahkan presentasi (desain) dari konten, sehingga kode menjadi lebih rapi dan mudah dikelola.
  • JavaScript
    JavaScript adalah bahasa pemrograman yang digunakan untuk menambahkan interaktivitas pada halaman web. Dengan JavaScript, web dapat menjadi dinamis, misalnya untuk membuat animasi, validasi form, atau pengambilan data dari server secara asinkron (AJAX). JavaScript memungkinkan pengalaman pengguna (user experience) yang lebih menarik dan interaktif.
